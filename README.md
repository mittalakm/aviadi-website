# AviAdi Enterprises Website

A simple static website for AviAdi Enterprises.

## Files

- `index.html` — the home page
- `styles.css` — site styling

## Viewing locally

Open `index.html` in any web browser, or serve the folder with a local web server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000 in your browser.
